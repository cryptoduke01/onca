# Brand — Onca

_Status: active_

## Product

Onca is a suite of Solana tools for ZeroClaw agents. The agent proposes; a human
disposes. Tools stay at T0 (read) and T1 (build unsigned requests). No spendable
keys. Name from Portuguese *onça* (jaguar), written plain as Onca.

## World

Night infrastructure. Pure black void with a soft cool-blue light field, like a
distant Solana mesh behind glass. Not warm brown, not cream, not indigo-slate
SaaS charcoal. Atmosphere lives in the light, not in mud.

## Typography

- **Display / UI:** Instrument Sans (self-hosted via `next/font`). Weight contrast
  does the hierarchy work. No second display face arguing with it.
- **Data:** system mono stack, only for genuine data (commands, addresses, tiers).

Avoid: Space Grotesk, Inter-as-identity, serif + italic accent default, tracked
caps as house voice.

## Voice

Short. Direct. Operator-facing. Simplified Technical English in docs. Marketing
copy states the custody rule in plain language: the agent proposes, a human
disposes. No hype, no "revolutionize", no fake metrics.

## Signature

1. The claw mark (three swept strokes) — keep, recolor to signal.
2. The working agent card — real interactive tool output, not a fake window.
3. Fail-closed transcript as proof artifact.

## Footer

Large composed wordmark, sparse links, real GitHub mark. Inspired by the
pols.dev colophon energy: one big name, little else, clear the cut.
